
# 滑动验证码缺口识别


##  **_slide_01_**

#### 识别范围
    网易滑块验证以及类似此类滑块

#### 识别原理
    根据opencv边缘检测, 用滑块的周围线条去匹配背景图中的滑块区域线条。
    对比图片误差比较大,只匹配线条成功率很高

##  **_slide_02_**

#### 识别范围
    极验系的滑块验证以及类似此类滑块

#### 识别原理
    根据原图与缺口图对比色差, 因为原图与缺口图只有滑块缺口位置颜色偏差比较大
